## âœ… Phase Validation

FORGE enforces quality gates at each phase:

- **MANDATORY requirements** block advancement until complete
- **RECOMMENDED items** show warnings but allow progression
- **Standards integration** enforces your project's existing tools
- **Clear guidance** on what's needed to advance

Example blocking conditions:
- Focus: Missing test scenarios, inadequate requirements
- Orchestrate: Insufficient task breakdown, missing test strategy
- Refine: Tests not passing, code review incomplete
- Generate: Build artifacts missing, documentation outdated
- Evaluate: Metrics not collected
